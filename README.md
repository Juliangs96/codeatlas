# CodeAtlas

Una enciclopedia abierta de programación. La idea es simple: que cualquiera pueda aprender o compartir lo que sabe sobre código, desde desarrollo web hasta robótica e inteligencia artificial.

---

## ¿Por qué lo hice?

Cuando empecé a aprender a programar me costaba encontrar recursos en español que estuvieran bien explicados y en un solo lugar. Siempre era un video por acá, un artículo por allá, y nada terminaba de cerrar. CodeAtlas es ese lugar que me hubiera gustado tener.

---

## ¿Qué tiene la página?

- **Inicio** — presentación del proyecto con imagen de fondo
- **Contenidos** — cards con las áreas disponibles: JavaScript, Python, PHP, Robótica e Inteligencia Artificial
- **Tutorial destacado** — un video introductorio para arrancar con HTML y CSS
- **Reseñas** — comentarios de usuarios de la comunidad
- **Comunidad** — explicación del propósito del proyecto
- **Contacto** — formulario funcional para colaborar o sugerir contenido

---

## Tecnologías usadas

- HTML5 semántico
- CSS3 (Flexbox, Grid, Media Queries)
- Google Fonts — Space Grotesk + JetBrains Mono
- Formspree para el formulario de contacto
- GitHub Pages para el hosting
